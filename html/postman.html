An overview of testing an API in POSTMAN

Step1: Create a collection in POSTMAN application

Considering an API in which we add a place , on successfully addittion we get place_id , using it for all the http methods

Click on collections icon and then on + symbol and provide a name to collection

Right Click on the collection , click on Add request

Provide a name to the Request , By default request selected will be GET , select the appropriate

POST Request

Lets see POST method, for POST we need to send the body

Provide the URI

Go to the body tab , select raw , select JSON as content Type and then provide the payload

Click on the Send button , we will get response if all the request parameters were valid .status code 200 will be seen

GET Request

Add one more request and select the GET method and provide the BASE URI

In GET whatever we have to send must be sent as the query params , provide it place_id we created using the POST request inside the Query params
