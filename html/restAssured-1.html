Lets see the Rest Assured Project setup

We must have java and eclipse installed in our machine

Create a maven project . Select File - New - Other - Maven - Maven Project - click Next - check mark create simple project [skip archetype selection] - provide groupId and artifactId - click on Finish

This creates a maven project and we can see pom.xml file

Add Dependencies

Go to maven repository and search for rest assured , testNG , hamcrest[used for assertions] dependencies and add to pom.xml file

Now we can continue with the rest assured topics
